## 一、什么是猪周期？—— 一个被生物规律锁死的经济现象

### 1.1 一句话定义

**猪周期**是生猪价格在供需关系驱动下呈现的周期性波动现象。一轮完整周期通常为 **3-4 年**（约 42 个月），包含"波谷 → 上涨 → 波峰 → 下跌 → 波谷"的完整循环。

### 1.2 为什么猪周期必然存在？—— 核心机制

猪周期存在的根本原因只有一个：**供给调节的滞后性**。这个滞后性由生猪的生物学规律决定，无法被人为消除。

```
你决定扩大生产 → 选留后备母猪（+4个月）→ 配种怀孕（+4个月）→ 仔猪育肥出栏（+6个月）
                 └──────────────── 总计约 14-18 个月 ────────────────┘
```

这意味着：**今天市场上的猪肉供应量，是由 18 个月前养殖户的决策决定的。** 当猪价高涨时，所有人同时决定补栏，但等这批猪真的出栏时（18 个月后），市场可能已经供过于求，价格暴跌。这就是猪周期最底层的逻辑。

### 1.3 理论框架：蛛网模型（Cobweb Model）

20 世纪 30 年代，经济学家 Ezekiel 和 Kaldor 提出了蛛网模型来解释这一现象：

```
价格(P↑) → 利润↑ → 产能扩张(Q↑) → 供过于求 → 价格(P↓) → 利润↓ → 产能缩减(Q↓) → 供不应求 → 价格(P↑)
└──────────────────── 一轮完整猪周期 ────────────────────────────────┘
```

## 二、中国猪周期的历史全景（2003—2026）

### 2.1 六轮完整周期概况

根据张立平等（2026）使用 H-P 滤波法对 2003-2024 年月度猪价数据的分析，中国已走完 **6 轮完整猪周期**，平均每轮约 **42 个月**。

| 周期 | 时间跨度 | 历时 | 价格高点 | 价格低点 | 峰谷落差 | 核心驱动因素 |
|:---|:---|:---:|:---:|:---:|:---:|:---|
| 第1轮 | 2003.03—2006.05 | 38月 | 9.59元/kg | 5.59元/kg | 4.6 | 非典疫情 + 禽流感 |
| 第2轮 | 2006.05—2010.04 | 47月 | 16.87元/kg | 5.96元/kg | 8.3 | 猪蓝耳病（PRRS） |
| 第3轮 | 2010.04—2014.04 | 48月 | 19.60元/kg | 9.62元/kg | 7.1 | 仔猪腹泻病 |
| 第4轮 | 2014.04—2018.05 | 49月 | 20.45元/kg | 10.57元/kg | 11.2 | 环保政策清退散户 |
| **第5轮** | **2018.05—2022.04** | **47月** | **37.14元/kg** | **11.32元/kg** | **22.6** | **非洲猪瘟（超级周期）** |
| 第6轮 | 2022.04—2024.04 | 24月 | 27.40元/kg | 13.30元/kg | 13.3 | 产能恢复 + 长期亏损 |
| **第7轮** | **2024.04—至今** | 进行中 | — | — | — | 效率提升 + 巨头内卷 |

### 2.2 关键特征变化

从周期参数的变化可以清晰看到行业变迁：

- **波动幅度越来越大**：第1轮峰谷落差 4.6，第5轮达到 22.6（非洲猪瘟极端值）
- **第6轮显著缩短**：仅 24 个月，不到此前均值的一半
- **季节性规律依然稳定**：每年呈现"V"型波动——年初和年末价格高（春节消费驱动），年中 6-8 月为低谷
- **价格中枢持续上移**：每轮周期底部价格逐步抬高（5.59 → 5.96 → 9.62 → 10.57 → 11.32 → 13.30 元/kg）

## 三、猪周期的四阶段运行机制

朱增勇 & 周亮（2023）将猪周期划分为四个阶段，以盈亏平衡点为关键分界线：

```
        猪价上行 ↑                     猪价下行 ↓
    ┌────────┬────────┐          ┌────────┬────────┐
    │ 理性   │ 非理性  │          │ 理性   │ 非理性  │
    │ 恢复   │ 扩张    │          │ 去化   │ 恐慌    │
    │ 产能   │ 产能    │          │ 产能   │ 出清    │
    └────────┴────────┘          └────────┴────────┘
      利润恢复    暴利驱动            持续亏损   踩踏出局
      谨慎补栏    疯狂扩产            有序退出   不计成本
```

**核心结论**：养殖户的决策是否"理性"，取决于三个因素：盈亏平衡点位置、盈亏程度、以及利润趋势是持续向上还是向下。

## 四、如何科学追踪猪周期？—— 三层指标体系

### 4.1 第一层：产能层（先行指标，领先约 10 个月）

| 指标 | 含义 | 使用方法 |
|:---|:---|:---|
| **能繁母猪存栏量** | 猪周期的核心引擎 | 连续 3 个月环比下降 >1%，才意味着真正的去产能开始 |
| 二元母猪价格 | 养殖户对未来的信心 | 母猪价格大跌 = 极度悲观，连"生小猪的母猪"都不留了 |
| 仔猪价格 | 补栏意愿 | 仔猪价格暴涨 = 大量补栏，预示未来供应增加 |

**为什么能繁母猪是最核心的指标？**

从能繁母猪到生猪出栏的时间链条：
```
能繁母猪配种 → 怀孕 4 个月 → 仔猪出生 → 育肥 6 个月 → 出栏
                    └────────── 总计约 10 个月 ──────────┘
```

农业农村部据此设计了调控机制：紧盯能繁母猪存栏量变化率，将调控关口前移了约 10 个月。

### 4.2 第二层：供应层（同步指标）

| 指标 | 含义 |
|:---|:---|
| 生猪存栏量 | 还有多少猪在栏里 |
| **出栏均重** | 压栏/抛售信号——均重上升 = 压栏惜售（供应后移），均重突降 = 抛售结束 |
| 屠宰量 | 即期市场供应量 |

### 4.3 第三层：需求与价格层（滞后指标）

| 指标 | 含义 |
|:---|:---|
| **猪粮比价** | 生猪价格 ÷ 玉米价格，行业盈亏"晴雨表" |
| 猪价 | 结果，不是原因 |
| 养殖利润 | 自繁自养/外购仔猪的头均盈亏 |

**猪粮比价预警等级**（国家发改委）：
- 5:1 以下 = 一级预警（全行业深度亏损，启动收储）
- 5:1 ~ 5.5:1 = 二级预警
- 5.5:1 ~ 6:1 = 三级预警
- 6:1 ~ 8.5:1 = 正常区间

## 五、影响猪周期的四大因素

张立平等（2026）系统总结了四大影响因素，它们相互关联、共同作用：

### 5.1 政策因素

| 政策类型 | 影响环节 | 核心作用 |
|:---|:---|:---|
| 养殖补贴政策 | 养殖、屠宰加工 | 刺激扩产（2012年存栏量创历史新高 5,067 万头） |
| 环保政策 | 饲料、养殖、屠宰 | 提高门槛、清退散户（2014-2016 年深度去产能） |
| 疫病防控政策 | 全产业链 | 限制调运、强制扑杀、分区域防控 |
| 猪肉收储政策 | 养殖、屠宰 | 托底信号（2026年4月已启动第二轮收储） |
| 生猪期货（2021年上市） | 养殖、屠宰 | 价格发现，但资本炒作加剧短期波动 |

**关键政策节点**：
- **2014年**：《环境保护法》+《畜禽规模养殖污染防治条例》
- **2016年**：划分"禁养区"，加速散养户退出
- **2018年**：非洲猪瘟后政策急转——从环保限制变为鼓励复产
- **2021年**：生猪期货上市大连商品交易所；设定正常保有量 4,100 万头
- **2024年3月**：正常保有量下调至 **3,900 万头**，下限从 95% 调整至 92%

### 5.2 规模化因素

**2023 年中国生猪规模化程度已达 68%**，这一变化深刻改变了猪周期的运行方式。

```
年出栏 500 头以上养殖户占比变化：
2014年: 0.55%
2019年: 0.68%
2022年: 0.94%（仍在快速提升）

TOP10 企业出栏占比：
2018年: 9.64%
2022年: 24.17%
2025年: >30%（占全国出栏量的41%）
```

**规模化如何改变周期**：
- **延长下跌期**：巨头融资能力强，亏损时不会轻易退出，产能去化缓慢
- **减缓上涨期**：补栏决策更审慎，短期不会一哄而上
- **波动频率加快**：资本和舆论影响下，一年内可能出现多次盈亏转换（2021.06-2023.04 不到两年出现 5 次盈亏转换）

**美国的前车之鉴**：美国生猪规模化进入深水区后，猪周期从 4 年拉长到 **7-11 年**，且价格波动幅度不降反升（近 5 轮平均 90%）。

### 5.3 疫情因素

疫情是能瞬间改写供需格局的最大外部冲击：

| 年份 | 疫情 | 影响 |
|:---|:---|:---|
| 2006 | 猪蓝耳病（PRRS） | 产能下降 19.8%，推动第2轮猪价暴涨 |
| 2010-2011 | 仔猪腹泻病 | 大量仔猪死亡，推动第3轮猪价上涨 |
| **2018.08** | **非洲猪瘟（ASF）** | **能繁母猪从 ~3500万头 降至 ~2035万头，猪价暴涨至 37.14元/kg** |

非洲猪瘟的冲击是全方位的：限制省际调运 → 产区过剩/销区短缺 → 强制扑杀 → 消费者恐慌 → 行业格局重塑。

### 5.4 成本因素

饲料成本（主要是玉米和豆粕）占总生产成本的约 **56.69%**，是决定养猪刚性成本的关键因素。仔畜费排第二。

当前（2026年4月）：饲料价格上涨（玉米连续27周同比上涨，豆粕连涨4周），而猪价跌至 10.28 元/kg，养殖端"卖一头亏一头"。

## 六、当前这一轮周期为何"前所未有"？

### 6.1 核心反常：能繁减少 ≠ 供给减少

这是当前周期最大的悖论：

```
2025年末：能繁母猪存栏 3,961 万头（较上年减少 116 万头）
但是：2025年猪肉产量 5,938 万吨（历史新高！）
      2025年生猪出栏 71,973 万头（同比 +2.4%）
```

**为什么？因为 PSY 大幅提升了。**

| 年份 | PSY（每头母猪年提供断奶仔猪数） |
|:---|:---:|
| 2017年 | 17.38 |
| 2025年 | 24.34 |
| 头部企业 | >29 |

生产效率提升 40%，意味着即使母猪减少了 100 多万头，总产出不降反升。

### 6.2 周期长度异常

| 统计方式 | 时长 | 历史均值 |
|:---|:---:|:---:|
| 从 2024.04 相对高点算起 | 下跌 24 个月 | 历史下跌均值 23 个月 |
| 从 2022 年末高点算起 | 下跌 **40 个月** | 严重超时 |
| 从 2020 年底高点算起 | 下跌 **64 个月** | 前所未有 |

### 6.3 当前状态（2026年4月）

- 生猪价格：**10.28 元/kg**（8年新低，海南等省甚至不到 8 元/kg）
- 猪粮比价：**4.23:1**（远低于 5:1 一级预警线）
- 自繁自养利润：**-339.99 元/头**
- 外购仔猪利润：**-183.95 元/头**
- 行业平均成本：**13 元/kg** 以上（牧原约 12 元/kg）
- 能繁母猪存栏：3,904 万头（2026年3月末）
- 调控目标：3,900 万头（仍略高于目标）

### 6.4 本轮周期的结构性变化

1. **成本内卷白热化**：牧原目标降到 11.5 元/kg，行业成本方差巨大，二线企业艰难求生
2. **出栏均重偏高**：128.51 kg/头（历史同期偏高），说明压栏普遍
3. **巨头继续扩张**：2025年出栏超百万头的猪企达 39 家（同比 +7 家），合计出栏 2.95 亿头（+24.5%）
4. **猪肉消费占比下降**：从 2018 年的 62.1% 降至 2025 年的 57.9%（牛羊鱼替代效应）
5. **去产能缓慢**：以当前淘汰速度，去化 300 万头可能还需 3 年

## 七、科学分析猪周期的框架

### 7.1 不应该做什么

- 不要简单套用"三四年一轮回"的历史规律
- 不要只看猪价（猪价是结果，不是原因）
- 不要轻信"历史底部，随时反转"的模糊预判

### 7.2 应该做什么

**第一步：跟踪先行指标**
- 每月关注农业农村部公布的能繁母猪存栏量
- 核心判断：是否出现**连续 3 个月以上的趋势性环比大幅下降**？

**第二步：设定可证伪的观察点**
- 如果连续 3 个月能繁母猪去化停滞甚至反弹 → "底部"判断失效
- 如果猪粮比回升至 6:1 以上 → 行业开始盈利
- 如果出栏均重开始趋势性下降 → 压栏结束信号

**第三步：区分结构性 vs 周期性**
- 周期性变化：随价格涨跌波动的供需调整（会回归）
- 结构性变化：规模化提升、PSY 提升、消费占比下降（不会回归）
